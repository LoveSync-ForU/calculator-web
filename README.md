# Kalkulator Modern dengan Animasi

Kalkulator sederhana yang dibuat dengan HTML, CSS, dan JavaScript yang memiliki animasi yang keren.

## Fitur

- **Desain Modern**: Tampilan glassmorphism dengan gradient background
- **Animasi Halus**: Hover effects, ripple effects, dan pulse animations
- **Responsif**: Bisa digunakan di desktop dan mobile
- **Keyboard Support**: Bisa menggunakan keyboard untuk input
- **Error Handling**: Menangani pembagian dengan nol

## Cara Menggunakan

1. Buka file `calculator.html` di browser Anda
2. Klik tombol untuk melakukan perhitungan
3. Atau gunakan keyboard:
   - Angka (0-9) untuk input angka
   - +, -, *, / untuk operator
   - Enter atau = untuk hasil
   - Backspace untuk hapus satu digit
   - Escape untuk clear

## Struktur File

- `calculator.html` - File HTML utama
- `calculator.css` - Styling dengan animasi
- `calculator.js` - Logika kalkulator

## Browser Support

- Chrome/Edge (terbaik)
- Firefox
- Safari
- Opera

## Customization

Anda bisa mengubah warna di file `calculator.css`:
- Gradient background: line 11-12
- Button colors: various btn classes
- Display colors: .display class
